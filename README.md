# Python Quests Project

This project is a collaborative group assignment completed as part of our introductory Python learning journey. The activity consisted of a structured set of programming challenges (“quests”) designed to gradually build understanding of core Python concepts through hands-on practice.

Each team member participated actively by taking responsibility for different levels of the quests. We worked together by dividing tasks, sharing solutions, reviewing each other’s code, and helping troubleshoot errors when they appeared. This teamwork approach helped us learn more efficiently and understand different ways of solving the same problems.

Throughout the project, we explored fundamental programming concepts such as:
- Printing output using `print()`
- Working with variables and data types (strings, integers, floats)
- Understanding basic logic and program structure
- Writing and executing simple Python scripts
- Using the terminal to run programs and manage files
- Practicing version control using Git and GitHub

The full set included 30 quests in total. As a group, we successfully completed 26 of them. However, we did not complete Quest 5, Quest 10, Quest 27, Quest 29, and Quest 30, which were skipped during the process due to time and progression choices.

Despite not completing every single quest, the project was very valuable in strengthening our understanding of Python basics and improving our collaboration and problem-solving skills. It also gave us practical experience working in a shared coding environment using Git workflows.

Overall, this assignment helped us build confidence in writing Python code and introduced us to real-world development practices such as teamwork, debugging, and version control.
